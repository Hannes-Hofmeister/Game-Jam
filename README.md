### Thema:
- Das Thema dieses Game Jams lautet: [Thema].
- Du bist eingeladen, ein Spiel zu entwickeln, das dieses Thema auf kreative und originelle Weise interpretiert.

### Regeln unseres Teams:
1. Du musst die Godot Engine verwenden, um dein Spiel zu entwickeln.
2. Du musst beim Game Jam unter Folgendender Seite angemeldet sein: https://stuve.uni-ulm.de/sparrows/hackathon-x-gamejam/anmeldung
3. Alle Assets müssen entweder von dir selbst erstellt oder aus lizenzierten Quellen stammen, die die Nutzung für Game Jam-Zwecke gestatten.
4. Das Spiel darf vor dem Beginn des Game Jams nicht entwickelt worden sein.
5. Bitte beachte alle geltenden Gesetze und Vorschriften bezüglich Urheberrecht und geistigem Eigentum.
